# badapple-py

-----

# Installation

## 1. Using github
```
-git clone https://github.com/Raj-1727/badapple-py
-cd ~/badapple-py
-makepkg -si
```
## 2. Using an AUR helper(currently not available)

```
yay -S badapple-py
# or
paru -S badapple-py
```
-----

# Usage

```
python badapple-py      # if running from the cloned repository
# or
badapple-py             # if installed from AUR
```

- Python
- Pillow
