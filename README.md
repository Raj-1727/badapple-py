# Installation

## 1. Using github
```
git clone https://github.com/Raj-1727/badapple-py
cd ~/badapple-py
makepkg -si
```
## 2. Using an AUR helper

```
yay -S badapple-py
# or
paru -S badapple-py
```
-----

# Usage

```
python badapple-py(badapple-py works too)      # if running from the cloned repository
# or
badapple-py                                    # if installed from AUR
```
---

# Controls
```
q - quit
spacebar - pause/unpause
```

# Dependencies

- **Python**
- **Pillow** - Image Processing
- **FFmpeg** - Audio
- **figlet** - ASCII Banner
- **lolcat** - Rainbow Gradients
